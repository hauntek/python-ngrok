# python-ngrok
基本上已经完善！并且24*7小时长时间工作，在期间我们多次尝试断网重连、渠道反复注册等，均无任何问题。

采用多线程异步处理，并发性能相当强悍！

# 运行环境
Python 2.7.9 或 Python 3.4.2 以上

# 运行方法
直接运行即可.

# 温馨提示
如果有小伙伴不想依赖环境运行，不妨可以试下PyInstaller，把py编译成可执行文件。
